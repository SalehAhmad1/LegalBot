WELSH STATUTORY INSTRUMENTS
2021 No. 264 (W. 70)
REPRESENTATION OF THE PEOPLE, WALES
The Candidate Election Expenses (Senedd Elections) Code of Practice 2021 (Appointed Day) Order 2021
Made
8 March 2021
The Welsh Ministers, in exercise of the powers conferred by paragraph 14(7)(b) of Schedule 7 to the National Assembly for Wales (Representation of the People) Order 2007(1) (“the 2007 Order”) make the following Order.
The Electoral Commission have prepared a draft Code of Practice and in accordance with paragraph 14(2) of Schedule 7 to the 2007 Order have submitted it to the Welsh Ministers for approval.
The Welsh Ministers have approved the draft Code of Practice and in accordance with paragraph 14(4) of Schedule 7 to the 2007 Order, a draft of the Candidate Election Expenses (Senedd Elections) Code of Practice 2021 was laid before Senedd Cymru on 20 January 2021.
In accordance with paragraph 14(6) of Schedule 7 to the 2007 Order, within a period of 40 days Senedd Cymru did not resolve not to approve the draft Electoral Commission’s Candidate Election Expenses (Senedd Elections) Code of Practice 2021, and the Code was issued in the form of the draft.
(1)
Paragraph 14 to Schedule 7 to the National Assembly for Wales (Representation of the People) Order 2007 (S.I. 2007/236) was inserted by the Senedd Cymru (Representation of the People) (Amendment) Order 2020 (S.I. 2020/1558 (W. 329)).Title and interpretation
1.—(1) The title of this Order is the Candidate Election Expenses (Senedd Elections) Code of Practice 2021 (Appointed Day) Order 2021.
(2) In this Order, “the Code of Practice” means the Candidate Election Expenses (Senedd Elections) Code of Practice 2021, a draft of which was laid before Senedd Cymru on 20 January 2021.Appointed day
2.  The day appointed for the coming into force of the Code of Practice is 9 March 2021.Julie James
Minister for Housing and Local Government, one of the Welsh Ministers
8 March 2021EXPLANATORY NOTE
(This note is not part of the Order)
This Order appoints 9 March 2021 as the day on which the Candidate Election Expenses (Senedd Elections) Code of Practice 2021 (“the Code of Practice”) comes into force.
The Electoral Commission prepared the Code of Practice and submitted it to the Welsh Ministers for approval. The Welsh Ministers have approved the Code of Practice, which was laid before Senedd Cymru on 20 January 2021.
The Senedd did not resolve not to approve the Code of Practice and as a result, the Welsh Ministers make this Order issuing the Code of Practice, as it was laid before the Senedd.
